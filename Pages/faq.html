Frequently Asked Questions

What should I do if I don't see the latest updates you've made to the website?

If it seems like the latest updates to the website are not applying for you, try pressing Ctrl + F5 on the page in question. This will refresh your browser cache. If this doesn't work, feel free to contact me for further assistance.

What is the purpose of this project?

Simple answer is I thought that GPRM's visual stat representation was amazing. So I thought I might as well make it for AniList since I use it a lot. Massive credit towards them as the default stat designs are from their project.

How can I contribute to this project?

There are several ways you can contribute to this project. If you're a developer, you can submit pull requests on GitHub with your proposed changes. If you're not a developer or prefer to contribute in other ways, you can join the Discord server and share your thoughts and ideas there. I appreciate all contributions, no matter how small!

How do I report a bug or request a feature?

If you encounter a bug or would like to request a new feature, please open an issue on the GitHub repository or discuss it in the Discord server. When reporting a bug, please include as much information as possible, such as the steps to reproduce the bug, the expected behavior, and the actual behavior. Screenshots or screen recordings can also be helpful. When requesting a feature, please explain in detail what you want and why you think it would be beneficial to the project. I appreciate your feedback and contributions!

How can I get help if I have a problem?

If you need help with anything, feel free to contact me. You can reach me on AniList or Discord. Alternatively, you can go to the #support channel in the Discord server, where I or others can assist you.
